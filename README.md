# Rust Calendar

A terminal calendar writen in Rust.

### Introduction 

A pet project to learn how to write CLI app in rust. Clone from `cal`.

<img src="./demo.png" alt="demo" />

### Usage

- Using left/right arrow key to go to prev/next month
- Press "q" to quit
